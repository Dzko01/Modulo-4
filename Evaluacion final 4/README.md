# Evaluación Final - Módulo 4: Programación Avanzada en JavaScript

## Objetivos de la Aplicación
El objetivo de esta aplicación es codificar un sistema de administración de usuarios en JavaScript, aplicando los conceptos clave aprendidos en el módulo:

Programación Orientada a Objetos (POO): Creación de una clase (AdministradorUsuarios) y métodos encapsulados para gestionar la información.
Manejo de Eventos y DOM: Controlar la interacción del usuario mediante botones HTML conectados de forma asíncrona mediante addEventListener.
Conexión de Datos: Consumo de un servicio web externo (API JSONPlaceholder) utilizando XMLHttpRequest para capturar y procesar la información de los usuarios.
